Privacy Policy & Security

🛡️ طبيعة الموقع / Website Nature

العربية: هذا الموقع هو مشروع تجريبي وتعليمي مطور لأغراض العرض والتعلم فقط. لا يتم جمع أو تخزين أو نقل أي بيانات شخصية إلى خوادم خارجية.

English: This website is a demo and educational project developed for showcase and learning purposes only. No personal data is collected, stored, or transmitted to external servers.

💾 تخزين البيانات / Data Storage

العربية: جميع البيانات التي تدخلها (الحسابات، المهام، الإعدادات) تُحفظ محليًا في متصفحك باستخدام تقنية localStorage. هذه البيانات لا تغادر جهازك أبداً.

English: All data you enter (accounts, tasks, settings) is saved locally in your browser using localStorage technology. This data never leaves your device.

🔒 الأمان / Security

العربية: لا توجد اتصالات بقواعد بيانات خارجية أو خوادم لجمع البيانات. الموقع يعمل بالكامل في متصفحك دون إرسال أي معلومات للخارج.

English: There are no connections to external databases or data collection servers. The website works entirely within your browser without sending any information externally.

🎯 الغرض / Purpose

العربية: هذا المشروع مصمم لعرض قدرات البرمجة والتطوير، ولتعلم تطوير تطبيقات الويب. يمكن استخدامه بأمان لتجربة إدارة المهام.

English: This project is designed to showcase programming and development skills, and for learning web application development. It can be safely used to experiment with task management.

⚠️ إخلاء المسؤولية / Disclaimer

العربية: بما أن البيانات محفوظة محليًا، قد تضيع إذا قمت بمسح بيانات المتصفح. ننصح بعدم استخدام معلومات حقيقية حساسة.

English: Since data is stored locally, it may be lost if you clear browser data. We recommend not using real sensitive information.
